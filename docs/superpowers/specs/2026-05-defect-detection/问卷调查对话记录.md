## 用户原始需求

> 请你给我描述这个项目的功能

### 回答

这是一个**煤图像识别分割项目**，基于 YOLOv8-seg 实现对原煤、矸石、异物进行实例分割。

**核心功能：**

1. **图像实例分割** — 识别并分割图像中的：
   - `coal`（原煤）
   - `gangue`（矸石）
   - `unknown`（异物）

2. **数据格式转换** — COCO JSON → YOLO txt 格式

3. **模型训练与推理** — 使用预训练模型 `yolov8s-seg.pt`

4. **远期目标** — 煤质软测量（水分、灰分、热值）

**B. 单独训练一个分类模型** — 准备一组"良品图片"和"残次品图片"，用 YOLOv8-cls 专门做二分类

## 用户偏好

> 你是专家你认为那种模式难度小容易实现

### 回答

**推荐方案 A：基于缺陷检测结果判定**

原因：

| | 方案 A（检测结果判定） | 方案 B（二分类模型） |
|---|---|---|
| 数据准备 | 直接用现有数据 | 需要重新整理分类数据集 |
| 标注成本 | 零额外标注 | 需重新标注良品/残次品 |
| 模型训练 | 1个检测模型 | 1个分类模型 |
| 实现复杂度 | 低（后处理加个判定逻辑） | 中（需要单独训练分类模型） |

**方案 A 的判定逻辑：**
```
if 检测到缺陷数量 > 0:
    判定为残次品
else:
    判定为良品
```

对于毕业设计来说，这个方案**开箱即用**，你的数据集已经是 YOLO 格式，训练完检测模型后，加一个后处理脚本就能输出二分类结果。

**核心逻辑：**

- 推理时：检测到任何缺陷（任意类别）→ 残次品；无缺陷 → 良品
